# Homebrew Tap

Homebrew formulae for [dortort](https://github.com/dortort) projects.

## Install

```sh
brew install dortort/tap/<formula>
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| [dgossgen](https://github.com/dortort/dgossgen) | Generate dgoss-ready test suites from Dockerfiles |
